# Django CRUD REST API

## Objective

Build a simple CRUD REST API using Django and Django REST Framework.

## Features

- Create Student
- Read Student
- Update Student
- Delete Student
- URL Routing
- REST API

## Technologies

- Python
- Django
- Django REST Framework

## API Endpoints

GET      /students/

POST     /students/

PUT      /students/<id>/

DELETE   /students/<id>/

## Run

python manage.py migrate

python manage.py runserver
