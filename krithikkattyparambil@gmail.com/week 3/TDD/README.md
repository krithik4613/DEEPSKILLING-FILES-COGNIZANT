# Test Driven Development (TDD)

## Objective

This project demonstrates the Test Driven Development (TDD) approach.

### TDD Cycle

1. Red
- Write a failing test.

2. Green
- Write the minimum code required to pass the test.

3. Refactor
- Improve the code while ensuring all tests continue to pass.

## Example

Calculator operations:
- Addition
- Subtraction
- Multiplication
- Division

## Commands

Run the tests using:

python test_calculator.py
